Question-1: What are the different ways to select an element in the DOM?

There are several methods available for selecting elements in the DOM, including querySelector(), querySelectorAll(), getElementsByTagName(), getElementsByClassName(), getElementById(), and getElementsByName().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, textContent is innerHTML gives us tag while innerText gives us only the text inside the tag and textContent gives us text content of an element and all its descendants.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is attached to a parent element instead of attaching event listeners to multiple child elements

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event, such as a click or keypress, propagates from the target element up to its ancestors in the DOM hierarchy (parent, grandparent, etc.)

Question-5: How do you create, add, and remove elements using JavaScript?

I can create element with createElement() methods. I can add element using aappendChild() methods and I can remove element using remove() methods.
